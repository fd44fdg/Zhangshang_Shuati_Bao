# 掌上刷题宝项目测试计划

## 测试概述

本文档提供了掌上刷题宝项目的全面测试计划，包括单元测试、集成测试和用户验收测试。

## 项目结构

- **admin-panel**: Vue 3 + Element Plus 管理后台
- **backend**: Node.js + Express + MySQL 后端服务
- **zhangshang-shuati-app**: uni-app 移动端应用
